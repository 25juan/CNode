# CNode

由React Native 开发的移动APP,数据接口由 `http://cnodejs.org/` 提供
## 使用教程

1. 下载项目
    ###### `git clone git@github.com:25juan/CNode.git`
1. 执行 yarn 命令安装项目运行所需要的包 
    ###### `yarn`
1. 运行程序
    ###### `react-native run-ios ` or `react-native run-android`
